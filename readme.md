# Projet React en TypeScript (Next.js)

Ce projet utilise Next.js avec React et TypeScript.

## Prerequis

- Node.js 20 ou plus recent
- npm

## Installation

```bash
npm install
```

## Lancer en developpement

```bash
npm run dev
```

Puis ouvre http://localhost:3000

## Build de production

```bash
npm run build
```

## Lancer en production

```bash
npm run start
```

## Structure de base

- app/layout.tsx: layout racine
- app/page.tsx: page d'accueil
- tsconfig.json: configuration TypeScript
